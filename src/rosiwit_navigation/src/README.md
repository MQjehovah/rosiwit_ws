# Diffbot Navigation Package

This package provides differential drive robot navigation capabilities.

## Directory Structure

- `src/navigation/` - Core navigation module (smooth single-point navigation)
- `src/obstacle_avoidance/` - Obstacle avoidance module
- `src/narrow_passage/` - Narrow passage traversal module
- `src/controller/` - Motion control module
- `include/diffbot_navigation/` - Header files
- `test/` - Unit tests
